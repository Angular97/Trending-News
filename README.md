 ## Contribution Guidelines üèó

 To start contributing, follow the below guidelines:

**1.** Fork this repository.

**2.** Clone your forked copy of the project.

```
git clone https://github.com/imhardikdesai/news-website.git
```

**3.** Navigate to the project directory :file_folder: .

```
cd news-website
```

**4.** Add a reference(remote) to the original repository.

```
git remote add upstream news-website.git
```

**5.** Check the remotes for this repository.

```
git remote -v
```

**6.** Create a new branch.

```
git checkout -b <your_branch_name>
```

**7.** Perfom your desired changes to the code base.

**8.** Track your changes:heavy_check_mark: .

```
git add .
```

**9.** Commit your changes .

```
git commit -m "Relevant message"
```

**10.** Push the committed changes in your feature branch to your remote repo.

```
git push -u origin <your_branch_name>
```

**11.** To create a pull request, click on `compare and pull requests`.

**12.** Add appropriate title and description to your pull request explaining your changes and efforts done.

**13.** Click on `Create Pull Request`.

**14** You did it! ü•≥ Wait for your submission to be accepted and your PR to be merged.
